# my-vue-admin 是前端代码 
动力港项目是一个充电桩的后台管理项目，里面包括了数据看板，充电站管理，电子地图，运营管理，报警管理，会员卡管理，招商管理，系统设置，个人中心等功能

运用的技术栈是： Vue3 + vue-router 4.x + pinia + TypeScript + Vite + element plus + axios + less + echarts + 高德地图api + TinyMCE富文本编辑器下面是部分页面截图
